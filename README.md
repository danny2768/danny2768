Software developer building full-stack systems from initial concept through deployment — turning coffee into code, and complex requirements into clean, maintainable architecture.

- 🔭 I'm currently a Software Developer at **Universidad Industrial de Santander**, building [UISARD](https://gestion.uis.edu.co/auth/#/auth/login).

- 💬 Let's talk Angular, TypeScript, NestJS, and building maintainable systems that scale — frontend, backend, or the architecture connecting them.

### 🚀 A few things I've shipped

- **[Diagnoodle](https://www.daniel-cobos.com/projects/diagnoodle)** — Clinical management platform, full-stack across Nuxt and Laravel.
- **[UISARD](https://www.daniel-cobos.com/projects/uisard)** — Centralized system for students, providers, and internal university operations, built in Angular.
- **[Walia](https://www.daniel-cobos.com/projects/walia)** — AI-powered WhatsApp assistant on NestJS, Prisma, pgvector, and the Vercel AI SDK.
